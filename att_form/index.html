  1. Na tag < input > com atributo type="text", para que serve o atributo "placeholder"?

    • O atributo serve para deixar uma mensagem de fundo da caixa de texo, sendo utilizado muitas vezes para indicar o que deve ser digitado.


  2. Na tag < input > com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

    • O atributo faz com que a caixa de seleção já esteja selecionado quando o site for carregado.


  3. Na tag < input > para que serve o atributo "value"?

    • Seria um valor definido dentro do input, informação já colocada no campo, mas que pode ser alterada


  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

    • O atributo "name" deverá ser utilizado, para que dentro de um mesmo grupo de opções, apenas um seja o escolhido.


  5. Para que serve a tag < label >? Para que serve o atributo "for" nessa tag?

    • A tag < label > seria um rótulo para o texto que precede a caixa de preenchimento do formulário, que ao receber um clique, direciona para o que o usário deverá preencher.
      Já o "for" é para se igualar ao "id" estabelecido do campo e assim se identificarem.


  6. Na tag < option > para que serve o atributo "selected"?

    • Para previamente selecionar a opção em que o atributo está.


  7. Nas tags de formulário para que serve o atributo "required"?

    • Definir a prioridade do campo, sendo ela obrigatório para o envio do formulário.


  8. Para que serve a tag < textarea >?

    • Seria um tipo de campo de texto que possui capacidade e tamanho maior. Utilizado para endereço, descrição, etc…


  9. Na tag < form > para que serve o atributo "action"?

    • Neste atributo devemos informar o nome da URL para a qual o conteúdo do formulário será submetido, se este atributo não for especificado, a URL corrente será usada.


  10. Na tag < form > para que serve o atributo "method"?

    • Serve para especificar o método de submissão usado:
      get - É o método padrão e faz com que o conteúdo do formulário seja anexado ao final da URL especificada no atributo "action"
      post - esconde os dados do usuário e manda diretamente pela rede, os dados não ficam visíveis e possui mais segurança


  11. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag < form >?

    • O atributo "post" deve ser utilizado para essa situação


  12. Qual valor deve ser informado no atributo "method" na tag < form > em um formulário que será usado para fazer upload de arquivos?

    • O atributo "post" também deve ser utilizado para essa ocasião


  13. O que as tags < fieldset > e < legend > produzem?

    • A tag < fieldset > seria para o agrupamento de campos do formulário em uma mesma caixa, podendo ser utilizado para dados de mesma categoria.
      Já a tag < legend >, seria o título exibido na caixa de agrupamento.
